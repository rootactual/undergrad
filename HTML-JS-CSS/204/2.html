Difference between Descendant and Child selectors

Semantic: "Descendant selector is select child of child of so forth child element where else Child selector is only of that child of specific element".
From way2tutorial.com

Syntactic (Descendant):
div p {
background-color: yellow;
}

Syntactic (Child):
div > p {
background-color: yellow;
}
